Importar biblioteca de Steam

Trae juegos desde tu cuenta de Steam y guárdalos en MongoDB y ChromaDB con su steamID. Si dejas vacía la API key, el servidor usará la variable STEAMWORKS_API_KEY si ya la configuraste.

Puedes pegarla aquí o dejar este campo vacío.
Cantidad máxima a importar por request.

Qué hace la importación

El servidor consulta tus juegos con la API oficial de Steam, descarga los detalles de cada app, los transforma al formato del proyecto y los indexa para búsqueda por nombre y similitud.

Si un juego ya existe por id, se omite para evitar duplicados.
